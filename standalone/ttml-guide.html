Use the BBC Subtitle Details extension

  1. Download the Chrome extension ZIP ↗ and unzip it.
  2. Open chrome://extensions, turn on Developer mode, select Load unpacked, and choose the extracted folder.
  3. Open or reload the BBC page containing the video. Start playback and enable subtitles.
  4. Open BBC Subtitle Details from Chrome’s extensions menu.
  5. Select Copy both for validator.
  6. Return here and paste into Import via Chrome extension. The PID and subtitle URL will be filled automatically.
  7. Choose Horizontal or Vertical, then select Validate TTML.

Subtitle URLs are signed and expire, so import them while they are current.
